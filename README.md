# Code Execution Visualizer

A lightweight, language-agnostic code execution visualizer built with HTML, CSS, and vanilla JavaScript. It simulates program flow using an abstract execution model and presents animated views for the current line, variables, and call stack.

## Features
- Language selection (Java, Python, JavaScript, Go)
- Run (auto-play) and Debug (step-by-step) modes
- Adjustable execution speed with Pause/Resume
- Animated visualization stage for line, variables, and call stack
- No external libraries or frameworks

## Usage
Open `app/index.html` in your browser.
